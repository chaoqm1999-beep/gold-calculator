# 黄金持仓盈亏计算器 V2 - 需求升级

**日期：** 2026-05-28
**版本：** V2.0

---

## 🔄 需求变更

### 1. 金价体系升级

**原方案：** 单一金价输入

**新方案：** 双金价体系
- **实时买入价** - 买入时的参考金价
- **实时卖出价** - 卖出时的参考金价

### 2. 趋势图表功能

**新增功能：** 金价趋势展示
- 实时（今日）
- 近1月
- 近3月
- 近半年
- 近1年

**数据来源：**
- 手动记录历史金价
- 或接入第三方金价API（待定）

### 3. 导航结构调整

**原方案：** 3个Tab页
- 持仓总览
- 交易记录
- 设置

**新方案：** 5个一级菜单（底部导航）
1. **首页** - 金价趋势 + 持仓总览
2. **买入** - 买入记录表单
3. **卖出** - 卖出记录表单
4. **记录** - 全部交易记录
5. **设置** - 导入导出

## 🎨 UI设计规格

### 金价展示卡片

```typescript
// 双金价卡片
interface PriceCard {
  buyPrice: number;      // 实时买入价
  sellPrice: number;     // 实时卖出价
  buyChange: number;     // 买入价涨跌
  sellChange: number;    // 卖出价涨跌
  updateTime: string;    // 更新时间
}
```

**样式：**
- 左右并排两个价格卡片
- 买入价：金色主题 `#D4AF37`
- 卖出价：银色主题 `#C0C0C0`
- 涨跌：红色↑、绿色↓

### 趋势图表

**数据结构：**
```typescript
interface PriceHistory {
  date: string;          // YYYY-MM-DD
  buyPrice: number;      // 买入价
  sellPrice: number;     // 卖出价
}
```

**图表库：**
- 使用 `recharts` 或 `echarts-for-react`
- 折线图展示双金价走势
- 支持缩放、拖拽

## 🔧 技术方案调整

### 新增依赖

```bash
npm install recharts
# 或
npm install echarts echarts-for-react
```

### 数据模型更新

```typescript
// 金价记录（新增）
interface PriceRecord {
  id: string;
  date: string;          // YYYY-MM-DD HH:mm
  buyPrice: number;      // 买入价
  sellPrice: number;     // 卖出价
  source: 'manual' | 'api';  // 来源
}

// Settings更新
interface Settings {
  currentBuyPrice: number;   // 当前买入价
  currentSellPrice: number;  // 当前卖出价
  currency: 'CNY';
  priceHistory: PriceRecord[]; // 历史金价
}
```
